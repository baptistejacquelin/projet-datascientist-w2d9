THP Project from Week 2 - Day 9
In this project, we were asked to do 2 exercices: one manipulating arrays, another manipulating hashes.
My files'names begin with '00_' or '01_', depending on which exercise they are used in (00 for the first one and 01 for the second one)
My project's architecture is the following: i have all my files (except this readme) in the directory 'lib'. Then i have split my code into 2 additional directories: 'main' and 'methods'.
In 'methods', you will find all my methods used for both exercises.
In 'main', you will find only variables, and calls to my methods.
